¿Cómo funciona el Sistema de Notificaciones?
  1. Creación Automática: Cuando ocurre un evento importante (nueva actividad, vencimiento de cuota, etc.), el sistema crea automáticamente una notificación.
  2. Envío por Email: La notificación se envía inmediatamente al email del usuario usando plantillas HTML profesionales.
  3. Almacenamiento: Se guarda una copia en la base de datos para consulta posterior en el panel web.
  4. Seguimiento de Estado: El sistema rastrea si la notificación ha sido vista y leída por el usuario.
